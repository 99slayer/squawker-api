#### Features
- RESTful API
- MongoDB object modeling using Mongoose
- User authentication/authorization with Passport.js
- Password hashing with Bcrypt
- Input validation using express-validator

#### Server Tech Stack
- NodeJS
- Express
- TypeScript
- MongoDB/Mongoose
